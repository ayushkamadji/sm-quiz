#### Repo for sitemate

##### Approach

- Client + server architecture
- Client: react + tailwind
- Server: Nest
- Db: sqlite

Client features:

- Issue schema:
  - id
  - title
  - description
- Page: List of issues (create button, delete button)
- Page: Create issue (title, description, create button)
- Page: Edit issue (title, description, update button)

Server features:

- API: Create issue
- API: Get all isssues
- API: Get issue by id
- API: Update issue
- API: Delete issue
